\section{Conclusion}

Until stage I of the project, we are able to generate results for the existing evaluation matrix, but we cannot conclude anything from it yet. The values seem to converge after some time in the comparison. Patrolling might not be affected by the communication device range or number of base stations. In future work, we will,
\begin{itemize}
\item Come up with a better evaluation matrix for the algorithm and analyze the results
\item After successful analysis, we plan to publish it in the journal
\item If time permits, we also plan to implement a minimal hardware setup.
\end{itemize}
